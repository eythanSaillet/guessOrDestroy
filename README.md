# Guess or DESTROY

Guess or DESTROY is a canvas game which use physic library and more to make you know some worship pieces of art.

# Rules :

Just try to guess what is the picture by typing the name with your keyboard ! Or wait for the wrecking ball to destroy the image to switch to the next image.

# Features :

- Typing
- Grab the wrecking ball or the pixels

# Ressources :

- Matter.js
- Paper.js
